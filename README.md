# Electron Todo App

A modern desktop todo application built with Electron and Node.js.

## Features

- ✨ Create, edit, and delete tasks
- 📁 Organize tasks by categories
- 🎯 Set priority levels
- 📅 Due date management
- 🌓 Dark/Light theme support
- 💾 Offline storage
- 🔔 Desktop notifications

## Technologies Used

- Electron
- Node.js
- HTML5
- CSS3
- JavaScript
- SQLite/JSON (for data storage)
